# Parkinson's Disease Detection API

Machine Learning-based REST API for detecting Parkinson's disease from voice features using 13 selected features after correlation filtering.

## Features

- **High Accuracy**: Random Forest Classifier with 92.31% accuracy
- **13 Voice Features**: Selected after correlation analysis
- **REST API**: FastAPI-based with automatic documentation
- **Batch Predictions**: Support for multiple samples
- **Health Monitoring**: Built-in health check endpoints
